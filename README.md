## Compile
Requires CMake (> 3.8) and a C++17 compiler.

```console
$ git clone https://github.com/stefanolusardi/clp.git
$ cd clp && mkdir build && cd build
$ cmake --build . --config Release
$ cmake --install . --prefix <ANYWHERE_YOU_LIKE>
```
